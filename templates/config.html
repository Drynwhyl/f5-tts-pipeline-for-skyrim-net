Semantic Chunking

Plans natural breath points for long Russian phrases and reserves extra frame budget for punctuation pauses. Single F5-TTS generation is treated as ref audio + generated audio.

Reference Audio Normalization

F5-TTS copies the loudness of the reference audio. If some refs are loud and others quiet, the generated voices will also be inconsistent. Normalization adjusts all ref audio to the same target level — quieter refs = more natural timbre and no clipping.

{{ config.ref_normalization.target_dbfs }}
